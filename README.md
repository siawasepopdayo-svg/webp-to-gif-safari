# WebP to GIF Safari

`index.html` を配信すれば、iPhone Safari で使える `WebP -> GIF` 変換ページになります。

## 仕様

- サーバーサイド変換なし
- 静止画 `WebP` に対応
- アニメーション `WebP` にも対応
- 複数ファイルを順番に変換
- 各ファイルごとに進捗バーを表示
- 変換後は各 `GIF` を個別に保存

## 使い方

1. `index.html` を Safari で開く
2. `WebP を選ぶ` から複数ファイルを選択
3. `まとめて変換` を押す
4. 各カードの `GIF を保存` から保存

## 補足

- iPhone でホーム画面に追加して使う場合は、`https` で配信してください
- GIF は 256 色に減色して生成するため、元画像より色味が少し変わることがあります
- 長いアニメーションや大きい画像は iPhone で時間やメモリを多く使います
